1. GP-SBO 출력으로 Path width 조정

- 사용공식 -

조정값[dB] = 20 x log10 (wG/0.7)

조정값[mW] = SBO출력[mW] x (wG/0.7)^2

* wG = 현재 path width 값[°]

2. LOC-SBO 출력으로 CRS width 조정

- 사용공식 -

조정값[dB] = 20 x log10 (wL/wLC)

조정값[mW] = SBO출력[mW] x (wL/wLC)^2

* wL = 현재 CRS width 값(°)

* wLC = LOC의 CRS width(울산공항 : 5.48°)
